# ICO-To-GIF

**Input Source: 8 bits per pixel ICO file with a single image.**

**Output Format: 8-bit Uncompressed GIF(87a)**

Commands for converting:
  1. make clean
  2. make
  3. make Proj.o
  4. ./ICO-To-GIF *source file name* *destination file name*
